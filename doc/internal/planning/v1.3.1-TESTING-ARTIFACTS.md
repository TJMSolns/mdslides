## BUG-002 Fix Verification ('S' Key Handler)

### HTML JavaScript Extract

**Keyboard Event Handler** (lines 18-21 of navigation JavaScript):
```javascript
document.addEventListener('keydown', (e) => {
  switch(e.key) {
    // ... other keys ...
    case 's':
    case 'S':
      e.preventDefault();
      window.open('speaker.html', 'speaker-view', 'width=1024,height=768');
      break;
  }
});
```

**Verification**: ✅ PASS
- Both lowercase 's' and uppercase 'S' handled
- Calls `e.preventDefault()` to prevent browser default behavior
- Opens speaker.html in new window with dimensions 1024x768
- Window named 'speaker-view' for proper focus management
