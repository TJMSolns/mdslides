## Technical Architecture

The codebase follows **clean architecture** principles:

`domain/` contains *pure functional* logic
`infrastructure/` provides **I/O adapters**
`cli/` wires everything together

All backed by ***96 passing tests***!

## Anticorruption Layer

External dependencies are **isolated** using the *ACL pattern*:

`FlexmarkAdapter` translates **Flexmark AST** → `FormattedContent`
`ThemeJsonAdapter` parses **JSON** → `Theme`
`HTMLRenderer` converts `SlideDeck` → **HTML**

Domain stays ***pure and testable***!
