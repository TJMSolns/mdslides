What is MDSlides?

MDSlides is a domain-driven, test-first presentation framework.

It converts markdown files into beautiful HTML slide decks.

Key features:

- Type-safe domain model

- Property-based testing

- Clean architecture

Getting Started

Simply write your slides in markdown format.

Use the CLI to convert them to HTML.

Then open the HTML file in your browser and present!
